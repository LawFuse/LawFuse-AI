# Ask 页面知识库集成功能说明

## ✅ 已完成的功能

### 1. 后端修改 (`backfont/api/ask.js`)

#### 新增功能
- ✅ 集成知识库检索功能
- ✅ 自动从 `knowledge-index.json` 中检索相关案例
- ✅ 将检索到的案例内容作为参考资料发送给 AI
- ✅ 返回参考文件列表给前端

#### 工作流程
```
用户提问 
    ↓
后端接收问题
    ↓
从知识库检索 Top 3 相似案例
    ↓
组装 Prompt（包含案例内容）
    ↓
发送给 AI
    ↓
返回 AI 回答 + 参考文件列表
```

#### API 响应格式
```json
{
  "code": 0,
  "answer": "AI 的回答内容...",
  "referenceFiles": [
    {
      "caseNumber": "(2016)京民申3726号",
      "category": "人格权纠纷+侵权",
      "preview": "案件内容预览..."
    }
  ]
}
```

### 2. 前端修改 (`vuefont/src/components/ask.vue`)

#### 新增功能
- ✅ 接收并存储参考文件信息
- ✅ 在 AI 回复下方显示参考案例卡片
- ✅ 美观的参考案例展示样式

#### 显示内容
每个参考案例卡片包含：
- 📚 参考案例标题
- 案例编号（案例1、案例2...）
- 案号（如：(2016)京民申3726号）
- 分类（如：人格权纠纷+侵权）
- 内容预览（前 200 字）

## 🎨 UI 效果

参考案例区域特点：
- 位于 AI 回答下方，用分隔线区分
- 每个案例卡片有左侧蓝色边框
- 案号使用等宽字体突出显示
- 分类信息用灰色小字显示
- 内容预览限制长度，保持简洁

## 📊 检索逻辑

### 检索算法
- 基于关键词匹配
- 计算用户问题与案例关键信息的重合度
- 按相关度评分排序
- 返回 Top 3 最相关的案例

### 优势
1. **提高回答准确性**：AI 可以参考真实案例
2. **增强可信度**：展示参考来源
3. **透明化**：用户可以看到 AI 参考了哪些案例
4. **学习价值**：用户可以阅读相关案例深入学习

## 🔧 配置选项

### 调整检索数量
在 `backfont/api/ask.js` 中修改：
```javascript
const similarCases = searchCases(message, 3); // 改为 5 则返回 5 个案例
```

### 调整内容摘要长度
```javascript
content.substring(0, 500) // 改为 1000 则显示更多内容
```

### 调整预览长度
```javascript
preview: content.substring(0, 200) // 改为 300 显示更多预览
```

## ⚠️ 注意事项

1. **知识库必须存在**：确保 `backfont/data/knowledge-index.json` 文件存在
2. **首次加载**：第一次调用时会加载知识库到内存，可能稍慢
3. **无相关案例**：如果找不到相关案例，AI 会仅基于专业知识回答
4. **性能优化**：知识库只加载一次，后续检索非常快

## 🚀 后续优化建议

1. **向量检索**：升级为真正的向量数据库，提高检索准确率
2. **案例详情**：点击参考案例可查看完整内容
3. **引用标注**：在 AI 回答中标注引用了哪个案例
4. **用户反馈**：允许用户对参考案例的相关性进行评分
5. **智能筛选**：根据问题类型自动调整检索策略
